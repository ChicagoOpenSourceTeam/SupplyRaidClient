The MyFirstProject example is a great place to start learning.

Inside is everything you need to create a very simple strange
app which demonstrates the Root (MyFirstProjectRoot, start here),
the Context (ExampleContext), which is where your bindings go, 
a Command (StartCommand), a View (ExampleView) and a Mediator (ExampleMediator).
